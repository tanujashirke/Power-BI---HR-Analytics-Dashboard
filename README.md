# 📊 HR Analytics Dashboard | Power BI

## Overview

This project presents an interactive HR Analytics Dashboard built using Power BI to analyze employee demographics, attrition trends, job satisfaction, education distribution, and departmental performance.

The dashboard helps HR teams identify workforce patterns and make data-driven decisions to improve employee retention and organizational performance.

## Dashboard Features

✅ KPI Cards
- Total Employees
- Attrition Rate
- Average Age
- Average Salary
- Average Years at Company

✅ Interactive Visualizations
- Attrition by Job Role
- Attrition by Gender
- Education Field Distribution (Treemap)
- Job Satisfaction Matrix
- Attrition by Marital Status
- Performance by Department

## Tools Used

- Power BI
- Power Query
- DAX
- Data Modeling

## Skills Demonstrated

- Data Cleaning
- Data Modeling
- DAX Measures
- Interactive Dashboard Design
- Business Insights
- HR Analytics

## Key Insights

- Attrition Rate: 16.1%
- Average Employee Age: 37 Years
- Average Salary: 6.5K
- Average Tenure: 7 Years
